## Installation

Via Composer

```json
{
    "require": {
        "kevinlebrun/colors.php": "0.*"
    }
}
```

    $ curl -s https://getcomposer.org/installer | php
    $ php composer.phar install

```bash
composer require sevenecks/ansi
```

## Usage

```php
<?php

require_once __DIR__ . '/vendor/autoload.php';

use SevenEcks\Ansi\Colorize;
```
